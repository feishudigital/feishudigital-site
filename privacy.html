Privacy Policy

Last updated: February 23, 2026

We believe in minimal data collection. Our browser extensions process data locally in your browser. We do not run analytics, do not track users, and do not sell data.

1. Who we are

Feishu Digital is operated by Kevin Lin, Grootsruhe 3, 20537 Hamburg, Germany. Email: feishudigital@gmail.com.

2. Our products

Feishu Digital builds browser extensions and web applications, including:

  • Arztrechnung Prüfer — Free web app for checking medical bills (GOÄ/GOZ)
  • JVEG Rechner — Web app for JVEG compensation claims
  • Readability Scorer — Chrome Extension for readability analysis
  • AI Chat Organizer — Chrome Extension for organizing AI conversations

This list may change as we add or discontinue products.

3. Data we collect

Browser Extensions (Readability Scorer, AI Chat Organizer):

  • All text analysis and data processing happens locally in your browser
  • We do not transmit, store, or have access to your content
  • Settings and preferences are stored locally using the browser’s storage API
  • License key validation (Pro users) sends only the license key to the Lemon Squeezy API — no user content

Static Web Apps (Arztrechnung Prüfer):

  • All calculations and data processing happen entirely in your browser (client-side)
  • No data is transmitted to any server
  • Uploaded files (PDFs, images) are processed locally and never stored or sent

Web Apps with Accounts (JVEG Rechner):

  • Account data (email, name) for authentication
  • Application data you enter (stored encrypted in the EU)
  • No data is shared with third parties except for payment processing (Lemon Squeezy/Stripe)

This Website (feishudigital.de):

  • Static site hosted on GitHub Pages
  • We do not use cookies, analytics, tracking pixels, or third-party scripts
  • GitHub Pages may collect basic server logs (IP address, browser type) — see GitHub Pages privacy

4. Payment processing

Payments are processed by Lemon Squeezy (Lemon Squeezy, LLC), who acts as Merchant of Record. Lemon Squeezy handles billing, taxes, VAT, and invoicing. We do not store credit card numbers or payment details. See Lemon Squeezy’s Privacy Policy.

5. Third-party services

  • Lemon Squeezy — Payment processing and license management
  • Supabase — Database and authentication for web apps (EU servers, Frankfurt)
  • GitHub Pages — Static website hosting
  • Cloudflare Pages — Static web app hosting
  • Fly.io — Web application hosting (Frankfurt, Germany)
  • Bunny Fonts — GDPR-compliant font delivery (no Google Fonts)

6. Your rights (GDPR/DSGVO)

As an EU resident, you have the right to:

  • Access your personal data
  • Rectify inaccurate data
  • Request deletion of your data
  • Object to processing or request restriction
  • Data portability
  • Lodge a complaint with a supervisory authority

To exercise any of these rights, email us at feishudigital@gmail.com.

7. Data storage and security

Browser extension data is stored locally on your device and never transmitted. Web application data is stored on EU-based servers (Supabase, hosted in Frankfurt, Germany). We use encryption in transit (HTTPS/TLS) and at rest where applicable.

We may update this policy from time to time. Changes will be reflected on this page with an updated date. For material changes, we will notify affected users via email or product update.
